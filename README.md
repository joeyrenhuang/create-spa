A way to initial our web project quicky, base on templates.
##### How to use
```
- web
  -  webpack / vite / rollup
  -  vue / react / angular
  -  coffeescript ?
  -  pug ?
  -  jsex ?
  -  theme support ?

- web init [template-name] [project-name]
```
##### Example
```
web init vue, web init react 
```
